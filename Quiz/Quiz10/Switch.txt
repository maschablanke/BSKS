Welche zwei Aussagen gelten für das Switches?


[X] Ein Router kann üblicherweise auch alle Funkionen eines Switchs ausführen.

[ ] Ein Switch leitet empfangene Ethernet-Frames stets an alle Interfaces (Schnittstellen) weiter.

[ ] Ein Switch kann üblicherweise auch alle Funkionen eines Routers ausführen.

[X] Ist einem Switch eine Ziel-MAC-Adresse unbekannt, so leitet er den weiterzuleitenden Frame über alle angeschlossenen Interfaces (Schnittstellen) weiter (außer der Quellschnittstelle).
